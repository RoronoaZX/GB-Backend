# Agent Prompts

This is a checklist for the agent to follow when creating a new feature:

### Conventions

- [] bug: Fix the error on the 'path/to/file.js' file.
- [] feature: Create a new feature on the 'path/to/file.js' file.
- [] refactor: Refactor the code fetching products.
- [] docs: Create a new documentation focus on blah blah.
- [] style: Make the card ui premium look.
- [] chore: Update the package to lates version
